
# NULS Docs Website

NULS Docs is a website that guides users to understand the NULS project and NULS technical documentation.

## Getting Started

Install dependencies:

```bash
$ git clone https://github.com/nuls-io/devsite.git
$ cd devsite
$ yarn install
```

Run server:

```bash
$ yarn run dev
```

Generate:

```bash
$ yarn run build
```
